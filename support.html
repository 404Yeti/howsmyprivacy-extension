Frequently Asked Questions

Does HowsMyPrivacy slow down my browser?

No. The extension only runs when you click the icon (or optionally on page load). The analysis is lightweight and takes less than 2 seconds. It doesn't continuously monitor or run in the background.

Do you collect any data or track me?

Absolutely not. HowsMyPrivacy collects zero data. Everything runs locally in your browser. No telemetry, no analytics, no external API calls, no data transmission. You can verify this by reviewing the open source code on GitHub.

What does my privacy score mean?

Your score (0-100) is calculated from 5 categories, each worth 20 points:

‚Ä¢ 80-100 = Excellent privacy protection
‚Ä¢ 60-79 = Good privacy with some improvements possible
‚Ä¢ 40-59 = Fair privacy, several issues found
‚Ä¢ 0-39 = Poor privacy, immediate action recommended

My score is low. Is that bad?

Not necessarily. Context matters. A low score means potential privacy issues were detected, but you should review the specific issues to decide which ones are important to you. Some tracking may be acceptable depending on your threat model.

Can I use this on Firefox?

Yes! Firefox support is available but installation is currently manual. Download the Firefox build from our GitHub releases and follow the installation instructions.

What permissions does the extension need?

HowsMyPrivacy requires these permissions to function:

‚Ä¢ cookies - Read cookie count (not content) to analyze privacy
‚Ä¢ storage - Store scan results locally
‚Ä¢ tabs - Access current page URL
‚Ä¢ webRequest - Detect tracking scripts
‚Ä¢ <all_urls> - Analyze privacy on all websites

None of these permissions are used to collect or transmit data.

Can I disable certain checks?

Yes! Click the Settings button in the popup to access 40+ configuration options. You can toggle individual privacy checks on/off, disable categories, and customize the appearance.

How accurate are the results?

HowsMyPrivacy performs real-time analysis based on what's observable in your browser. Results are accurate for detected issues, but some privacy threats (like sophisticated fingerprinting) may not be detectable. Use results as a guide, not an absolute measure.

The extension isn't working. What should I do?

Try these troubleshooting steps:

1. Refresh the webpage and click the extension icon again
2. Disable other extensions temporarily to check for conflicts
3. Clear browser cache and reload
4. Check the browser console (F12) for error messages
5. If still not working, report the issue

Can I contribute to the project?

Yes! HowsMyPrivacy is open source. You can contribute code, report bugs, suggest features, or improve documentation. See the Contributing Guide for details.

Yes. HowsMyPrivacy is:

‚Ä¢ Zero data collection - nothing leaves your browser
‚Ä¢ No external dependencies - no third-party code
‚Ä¢ Community reviewed - transparent development on GitHub

How do I uninstall the extension?

To remove HowsMyPrivacy:

1. Go to chrome://extensions/
2. Find "HowsMyPrivacy"
3. Click "Remove"
4. Confirm deletion

All local data (scan results, settings) will be deleted automatically.

Common Issues & Solutions

  1. Extension shows "Loading..." forever - Refresh the page and try again. Some sites block content scripts.
  2. Score doesn't update - Click the refresh button in the popup to run a new scan.
  3. Settings don't save - Check if you have enough storage space. Clear old scan history if needed.
  4. Badge doesn't show on icon - Enable "Show badge indicator" in Settings.
  5. Detailed report is blank - Run a scan first, then click "View Detailed Report".
